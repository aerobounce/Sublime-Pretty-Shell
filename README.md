- Fast Formatting
- Syntax Checking
- Inline Syntax Error Popup

## Install

> [Available via Package Control][packagecontrol]

1. `Package Control: Install Package`
2. Install `Pretty Shell`


## Requirements

[`shfmt`][shfmt]

> **Command** is the name of the command you can use for **Key-Bindings**.

| Caption                | Command        |
| ---------------------- | -------------- |
| `Pretty Shell: Format` | `pretty_shell` |
